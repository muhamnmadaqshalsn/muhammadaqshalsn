Apa Itu Komunikasi Multimedia?

komunikasi yang melibatkan penggunaan berbagai media seperti teks, gambar, suara, video, dan animasi untuk menyampaikan informasi, melalui berbagai saluran dan format media yang berbeda,  yang dapat dilakukan melalui berbagai platform seperti internet, televisi, radio, dan media sosial.

Komunikasi Multimedia

Text

Tulisan/karakter yang digunakan untuk pertukaran informasi. Unsurnya yaitu, kode ASCII, font, warna, ukuran, dan tipe text

Audio Digital

Audio digunakan untuk menyampaikan informasi, emosi, dan suasana. Terdapat 3 jenis audio, mono-stereo, analog-digital, dan bit depth-sample rate. Dalam audio filtering kita dapat menggunakan bantuan aplikasi Audacity

Animasi

Animasi adalah sebuah gambar atau objek dimanipulasi untuk dapat bergerak. Kita dapat membuat animasi simple dengan menggunakan  Scratch dan PowerPoint

Image Processing

Gambar dapat direprentasikan dalam bentuk pixel dan vector. Terdapat banyak format gambar, contohnya GIF, JPG, PNG, TIF, dan lainnya. Gambar dapat dikonversi format, diperbaiki kualitasnya, dan dikompres

Video Editing

Dalam sebuah video kita dapat mengedit agar terlihat lebih memanjakan mata, contohnya dengan mengatur brightness, saturasi, dan lainnya.​ Kita dapat menambahkan teks, menggabungkan/memotong video, dan menghilangkan latar belakang pada video

Membuat Website

Kita dapat menyimpar informasi menggunakan website dengan sintak HTML dan CSS yang kemudian dihosting. Kita dapat membuat server windows menggunakan Wampserver dengan menampilkan informasi dari HTML yang telah kita buat
